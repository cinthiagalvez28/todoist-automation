To view a specific report, provide the report folder name with date and time. You can find this information in the allure-reports folder. Then, run:

Example:
```
npx allure open allure-reports/report_2026-04-20T15-33-07-659Z
```

If you want to view the latest report from the latest execution, run:

```
npm run report
```

In order tu run eslint, execute:

```
npm run lint
```
